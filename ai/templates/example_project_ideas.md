# Example Project Ideas

## Beginner Friendly

- World Cup Predictor
- Trivia Game
- Workout Generator
- Recipe Recommender
- Daily Journal AI

## Slightly More Advanced

- AI Study Assistant
- Movie Recommendation App
- Bird Sighting Tracker
- Habit Tracker
- Local Event Discovery App

## Important

Keep projects:
- small
- incremental
- simple

The goal is learning orchestration and specs,
not building a startup.
