# Domain Glossary

Use this file to maintain shared terminology.

Consistent terminology significantly improves:
- communication
- AI consistency
- implementation quality

| Term | Meaning |
|------|---------|
| Template | A predefined feedback structure the generated output must conform to |
| Transcript | The raw text record of the interview |
| Notes | The interviewer's few lines of rough observations |
| Feedback | The generated, template-conformant write-up, ready to copy into Ashby |
| Recommendation | The decision output: pass / no pass / talent pool |
| Role Requirements | Role summary / requirements used to inform the recommendation |
| Talent Pool | "Keep for future roles" outcome — not this role, but worth retaining |
| Ashby | The external ATS where finished feedback is pasted |
| Interviewer | The primary user; provides notes + transcript, receives feedback |
